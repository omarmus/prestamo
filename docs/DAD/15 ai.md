# 1. Objetivo

Centralizar toda la inteligencia artificial de la plataforma.

Ningún módulo consumirá directamente OpenAI, Claude o cualquier otro proveedor.

Toda llamada pasará por AI Gateway.

# 12. Reglas

Nunca llamar directamente un proveedor.

Todo pasa por AI Gateway.

Todo prompt tiene versión.

Toda respuesta registra costo.

Todo modelo tiene fallback.

Todo módulo define presupuesto.

Todo embedding tiene versión.
